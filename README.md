# capacitor-mapboxnav

capacitor mapbox navigation ndk

## Install

To use npm

```bash
npm install capacitor-mapboxnav
````

To use yarn

```bash
yarn add capacitor-mapboxnav
```

Sync native files

```bash
npx cap sync
```
